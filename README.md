# 밸런싱봇
STM32와 임베디드 제어를 활용해 개발한 자율 밸런싱 로봇
## 📌 프로젝트 개요
- 수행 기간 : 2026.04.03 ~ 2026.04.14
- 팀원 : 김광현, 위오현, 최민지
- 배경 : 임베디드 제어 시스템을 깊이 있게 이해하기 위해 본 프로젝트를 선택하였습니다. 균형을 잡기 위한 환경에서 PID 제어 시스템의 동작 원리를 파악하여 응답성을 확보하고, PWM 제어 알고리즘을 활용해 실시간으로 모터 출력을 조정하는 효율적인 로봇 제어 소프트웨어를 구축하고자 하였습니다.
- 목표 :
  - 실시간성 확보
  - 정밀 모터 제어
  - 자세 유지 안정
- 주요 기능 :
  - 로봇의 기울기 및 자세 측정
  - 실시간 밸런싱
  - 좌우 모터 속도 및 방향 제어
  - 표정 출력
  - 블루투스 통신 기능
